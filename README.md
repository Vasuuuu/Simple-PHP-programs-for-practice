# Simple-PHP-programs-for-practice
## To run this program 
* Install ***Xampp***
* Go to drive where Xampp is located 
* Open Xampp
* Go to ***htdocs***
* make a file like ***Filename.php***
* open that file in code editor
* copy this code
* save 
* open browser 
* type ***localhost/Filename.php***
